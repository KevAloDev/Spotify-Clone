🎵 Spotify Clone App

Un clon funcional de Spotify creado con HTML, CSS y JavaScript , que simula la experiencia de streaming de música con una interfaz moderna y responsiva. Ideal para demostrar habilidades en frontend, diseño de interfaces y desarrollo web interactivo.

🚀 Características Principales

  🎧 Reproducción de música simulada
  🔍 Búsqueda y filtrado de canciones
  📂 Gestión de listas de reproducción (local)
  ❤️ Funciones de favoritos
  🖥️ Interfaz responsiva para desktop y móvil
  ⚡ Animaciones y transiciones fluidas para mejorar la experiencia de usuario

🛠️ Tecnologías Utilizadas

  HTML5 – Estructura semántica y accesible
  CSS3 – Diseño responsivo, flexbox, grid y animaciones
  JavaScript (ES6) – Funcionalidades dinámicas y reproducción de audio
  Herramientas de desarrollo: Visual Studio Code, Git, GitHub
